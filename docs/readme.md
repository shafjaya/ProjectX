# ProjectX

A personal morning todo dashboard built with Claude Code. Uses the Eisenhower matrix to organise tasks by urgency and importance, helping track momentum through the day.

## What it does
- 2x2 Eisenhower matrix: Do First, Schedule, Delegate, Drop
- Check off tasks throughout the day to track progress
- Mind-heavy flag (red dot) to surface mentally demanding tasks
- Real-time Done/Left counts per quadrant
- Completed tasks stay visible (faded/struck) — last 3 per quadrant shown inline, older ones in a trash drawer
- Add tasks with quadrant selection, mind-heavy toggle, Enter key support

## How to run
Open `index.html` in a browser. No build step, no dependencies.
